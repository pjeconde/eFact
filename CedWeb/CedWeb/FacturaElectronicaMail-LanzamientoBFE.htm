Factura Electrónica para el uso del Bono Fiscal. (solución web)
Ya se encuentra disponible en nuestro sitio, el ingreso de facturas electrónicas para bienes de capital, informática y telecomunicaciones, según lo instrumenta la R.G. 2557.

Los fabricantes de bienes de capital, informática y telecomunicaciones que poseen un establecimiento industrial radicado en el país deberán emitir la factura electrónica para poder utilizar los bonos fiscales en el pago de impuestos nacionales.

Hemos incorporado algunos cambios en nuestro sitio, siendo compatibles para facturas comunes y también para las utilizadas en Bono Fiscal.

Consulte con su operador de Interfacturas si ya tienen habilitado el PUNTO DE VENTA para BONO FISCAL y averiguar cual es ?.

Informar el Nro. de PUNTO DE VENTA para BONO FISCAL en nuestro sitio.
Si es usuario de nuestro Servicio Web (Ingresando las facturas en nuestro portal), debe informar el PUNTO DE VENTA para BONO FISCAL de la siguiente manera:
>> Cambios en la configuración de los Datos del Vendedor.

 En nuestra página de Factura Electrónica, seleccionar la opción 2 (Configure los datos del Vendedor)


Posteriormente, cuando se visualice la página de datos del Vendedor, ejecute los siguientes pasos:

 


 >> Cambios incorporados al Detalle de Artículos.

Los nuevos campos son:
- Código Producto Vendedor:
Código interno que tiene el vendedor para identificar el producto. Este campo no es obligatorio.
- Código Producto Comprador (Nomenclador):
Código interno con el que el comprador identifica el producto. Para Bono fiscal este campo es obligatorio y se debe infomar el código nomenclador de producto de la secretaría de industria. Para facturas comunes no es obligatorio.
- Gravado / Exento / No gravado:
Marca que indica si el producto es gravado, no gravado o exento. Para bono fiscal si informa Alícuota IVA en 0 este campo es obligatorio. Caso contrario es opcional.
- Cantidad:
Cantidad expresada numéricamente. Para bono fiscal debe ser mayor a cero.
- Unidad:
Unidad de medida de la cantidad informada. Para Bonos Fiscales este campo es obligatorio.
- Precio unitario:
Precio unitario del producto de la línea actual expresada en $. Para bono fiscal si moneda es $ (Pesos) es obligatorio.
-  Alícuota IVA %:
Alícuota de IVA que se le aplica al producto (99.99).Para bono fiscal este campo es obligatorio.

>> Otros cambios incorporados: Comprobantes de Referencia.

Se incorpora al ingreso de comprobantes una grilla “Comprobantes de Referencia” para que pueda registrar Órdenes de compra, Remitos, Facturas de referencia o Notas de credito relacionados.

Los nuevos campos son:
- Código de referencia:
Debe seleccionar de la lista.
- Número de referencia:
Información del documento relacionado. Acepta únicamente números.
